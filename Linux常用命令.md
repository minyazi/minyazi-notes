## <a name="anchor1">Linux如何重启网络？</a>
```
RedHat：
（方式一）service network restart
（方式二）/etc/rc.d/init.d/network restart
SUSE：
（方式一）service network restart
（方式二）rcnetwork restart
（方式三）/etc/rc.d/network restart
```
